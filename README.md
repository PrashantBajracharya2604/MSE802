# MSE802 : Quantum Computing Activities.

## Overview
This repository contains all activities for the MSE802 course Quantum Computing. It uses Qiskit framework. It includes quantum gates, Bloch Sphere, etc.

### Activities done.
Quantum Gates - Hadmard (H) gate, Phase (S)(T) gates, flip gates (X)(Y)(Z).  
Bloch Sphere.  


## Technologies
* Framework : Qiskit
* Language : Python 3.14.4
* Environment : Jupyter Notebook
* Simulator : AerSimulator
